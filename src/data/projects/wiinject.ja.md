Wiinject was a project spawned out of necessity while working on [_Heiretsu_](/#/my-projects/haroohie). Originally making use of a .NET assembler project
in order to assemble ASM and C files into Riivolution patches, that approach was deemed too fragile and the project was restyled after NitroPacker's LLVM + ninja
approach (though it uses devkitPPC's gcc for the time being).
