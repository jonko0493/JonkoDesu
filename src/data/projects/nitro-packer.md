Very early on in the [_Chokuretsu_ project](/#/my-projects/haroohie), [Ermelber](https://www.ermiisoft.net) sent me a zip file with a Windows executable
called "NitroPacker.exe" which I gleefully ran. (Always run executable files people send you on Discord, kids!) He also, separately, set me up with a system
for applying ASM patches to the _Chokuretsu_ ROM. These two things eventually were combined and open-sourced to create NitroPacker.

NitroPacker is an open source, cross-platform utility and .NET library for packing and unpacking Nintendo DS ROMs that can also can apply ASM hacks to both ARM9 and overlay files.
For more information, check out the [README on GitHub](https://github.com/haroohie-club/NitroPacker/)!
